# Secrypto 项目需求文档

## 项目概述
Secrypto 是一款 Windows 11 桌面应用，用于替换系统自带剪贴板，同时集成轻量级密码管理器。基于 Tauri 2.0 + Svelte 5 构建，追求低资源占用与原生 Win11 体验。

## 功能需求

### 1. 剪贴板管理器

| 需求 | 优先级 | 说明 |
|------|--------|------|
| 自动记录剪贴板 | P0 | 监听系统剪贴板变化，自动抓取复制内容（文本 + 图片） |
| 历史列表展示 | P0 | 时间降序排列所有记录卡片，显示预览 + 时间戳 |
| 存储期限设置 | P0 | 用户可配置保留期限：1天 / 3天 / 5天 / 永久 |
| 过期自动清理 | P0 | 后台定时检查并删除过期条目 |
| 全局热键唤起 | P0 | 自定义快捷键唤起主窗口（默认 Win+V） |
| 单击粘贴 | P0 | 点击卡片将内容复制到剪贴板并粘贴到前台窗口 |
| 置顶功能 | P1 | 重要记录可置顶，始终显示在列表顶部 |
| 删除功能 | P0 | 手动删除不需要的记录 |
| 搜索功能 | P0 | 按文本内容实时搜索过滤 |
| 去重 | P1 | 连续相同内容不重复记录 |
| 系统托盘 | P0 | 后台运行时显示托盘图标，右键菜单控制 |

### 2. 密码管理器

| 需求 | 优先级 | 说明 |
|------|--------|------|
| 主密码锁 | P0 | 首次使用设定主密码，每次打开需验证 |
| AES-256 加密 | P0 | 所有密码条目加密存储 |
| 增删改查 | P0 | 管理密码条目的完整 CRUD |
| 密码字段 | P0 | 网站名 / URL / 用户名 / 密码 / 备注 / 标签 |
| 一键复制密码 | P0 | 点击复制按钮将密码写入剪贴板 |
| 密码生成器 | P1 | 可调长度 + 字符类型，生成随机强密码 |
| 搜索过滤 | P1 | 按名称、URL、标签搜索 |

### 3. 系统需求

| 需求 | 说明 |
|------|------|
| 操作系统 | Windows 11（Build 22621+） |
| 资源占用 | 后台常驻内存 < 50MB |
| 开机自启 | 安装时可选择，设置中可开关 |
| 安装包 | MSI 安装包 |

## 用户交互流程

### 复制 → 记录 → 粘贴 流程
1. 用户在任何应用中选择文本/图片 → 按 Ctrl+C
2. Secrypto 后台监听捕获内容 → 存入本地 JSON
3. 用户按 Win+V（全局热键）→ 弹出历史面板
4. 在历史面板中浏览 / 搜索 / 选择
5. 点击卡片 → 内容写入剪贴板 → 模拟 Ctrl+V 粘贴 → 面板隐藏

### 密码管理器流程
1. 在主窗口中切换到「密码本」标签页
2. 输入主密码解锁（首次需设置）
3. 浏览、搜索、添加、编辑、删除密码条目
4. 点击密码旁的复制按钮 → 密码写入剪贴板

## 非功能需求
- 响应速度：剪贴板变化 500ms 内完成记录
- 并发安全：JSON 文件读写使用文件锁
- 数据安全：密码库全程 AES-256-GCM 加密
- 国际化：优先中文界面，保留英文资源文件
