## 数据流

### 剪贴板记录流程
```
[用户 Ctrl+C] → [Windows 剪贴板]
     → [AddClipboardFormatListener 通知]
     → [Rust: 读取剪贴板内容]
     → [去重检查]
     → [序列化 JSON → 写入文件]
     → [Tauri Event 通知前端刷新]
     → [Svelte: 更新列表 UI]
```

### 粘贴流程
```
[用户 Win+V] → [全局热键触发]
     → [Tauri: 显示主窗口]
     → [用户点击卡片]
     → [Rust: 内容写入系统剪贴板]
     → [Rust: 模拟 Ctrl+V 到前台窗口]
     → [Tauri: 隐藏主窗口]
```

## 安全架构

### 密码库加密
```
[主密码] ──Argon2id──→ [256-bit 密钥] ──AES-256-GCM──→ [密文]
                                    ↑
                              [随机 96-bit nonce]
```

### 内存安全
- 主密码输入后立即派生子密钥，原始密码清零
- 解密后的密码在 Rust 侧使用 `zeroize` 清理
- 前端仅持有临时解密结果，切换标签页即清除
